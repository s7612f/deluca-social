# De Luca — Brand Guide 2026

Project code: `dlc`

---

## Direction

Contemporary Italian restaurant. Elevated but warm. Think: modern trattoria in Soho, not red checkered tablecloth. The brand should feel editorial, premium, and rooted in Italian craft — without leaning on tired clichés.

---

## Colour Palette — Terracotta-led

Terracotta replaces the dated crimson. It still reads "Italian" but feels modern, warm, sun-baked rather than heavy/wine-coloured.

| Name          | Hex       | Role       | Use                                   |
|---------------|-----------|------------|---------------------------------------|
| Terracotta    | `#C4622D` | PRIMARY    | Hero colour — main brand identity     |
| Terra Scuro   | `#A04E1F` | ACCENT     | Deeper terracotta — text on light bg  |
| Terra Chiaro  | `#E08A50` | HIGHLIGHT  | Lighter terracotta — gradients, glow  |
| Nero          | `#18170F` | BASE       | Text, dark backgrounds                |
| Avorio        | `#F4EDE3` | GROUND     | Light backgrounds, reversed text      |
| Oro           | `#C9A96E` | DETAIL     | Premium accents, rules, borders       |
| Fumo          | `#6B5F55` | SECONDARY  | Muted captions, secondary text        |

**In Adobe Express:** Brand > Colours — set **Terracotta** as primary, Nero as base, Avorio as ground.

**Rule:** One terracotta tone per design. Don't mix Terracotta + Terra Scuro + Terra Chiaro in the same graphic.

## Typography

### Display — Cormorant Garamond
Use: headlines, dish names, Italian phrases, pull quotes
Weight: **Bold (700)** or SemiBold — NOT light. Size: large.
Adobe Express: search "Cormorant Garamond"

### Body / UI — DM Sans
Use: captions, hashtags, dates, CTAs, labels
Weight: Regular 400, Medium 500
Adobe Express: search "DM Sans"

**Rule:** Never use thin/light Cormorant at small sizes. If it's under 24px, use DM Sans instead.

## How to Set Up in Adobe Express

1. **Branding > Create brand kit**
2. Add colours using hex codes above
3. Add fonts: Cormorant Garamond (Display), DM Sans (Body)
4. Upload logo: use files in `artwork/brand_assets/logos/`
5. Upload brand images: use food photography from `footage/photos/raw/2026/food_pics/`
6. Upload elements: SVG files from `artwork/rebrand_2026/elements/`

## Usage Rules

- One dominant brand colour per post
- Cormorant Garamond headlines: always large, always bold
- Never use more than 2 colours in a single graphic
- Food photography: warm grade preferred, not blue/cool
- Avoid script fonts — they read as dated
- Breathing room: generous padding, don't fill the frame

---

## What's different from the old brand

| Old                          | New                                   |
|------------------------------|---------------------------------------|
| Crimson + cream only         | Full 6-colour palette with salvia + oro |
| Thin light-weight serifs     | Bold editorial use of Cormorant       |
| Lots of copy, small text     | Big type, less words                  |
| Traditional ornate borders   | Clean geometric brackets + rules      |
| Script/decorative typography | Cormorant Bold + DM Sans only         |
