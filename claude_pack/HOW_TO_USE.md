# How to use this folder

1. Drag the entire `claude_pack/` folder into a Claude conversation (claude.ai supports folder uploads)
2. Just ask. Examples:
   - "make me a feed post about the new lasagna"
   - "give me 3 captions for the terrace opening"
   - "do a 5-slide carousel for the 20th anniversary"
   - "write a Reel script for tonight's service"
   - "plan next week's content"
   - "I need a story for tonight, make it"
   - "make me something — engagement is dropping"

3. Claude reads `CLAUDE.md` automatically as context and outputs SVG / captions / Canva steps / Reel scripts / etc — already on-brand.

You don't need to explain the brand, the colours, or the rules. They're all in `CLAUDE.md`.
